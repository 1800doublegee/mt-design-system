# Made Thought — Slide Pattern Reference

A quick-reference guide to all 33 slide patterns. Use this to select the right template for any content brief.

---

## How to select a template

1. **What is the primary content type?** Numbers → Stats. Framework → Matrix or Brand Framework. Story → Text Intro or Title+Body. Evidence → 6-Card Grid.
2. **How many items?** 1–2 items → Split or Today-Tomorrow. 3 items → 3-Col or Stats. 4 items → 2×2 Matrix or 4 Recommendations. 6 items → 6-Card Grid.
3. **Is this structural or content?** Structural (opens/closes/punctuates) → Dark template. Content → Light template.

---

## Pattern index

### COVER
**When:** Always the first slide of a deck.
**Content:** MT logo + deck title (Display/XL or Mega) + subtitle (Eyebrow) + optional date.
**Theme:** Dark.
**Rule:** The title should be a declarative statement, not a description. "Desire by Design" not "Brand Strategy Presentation."

---

### SECTION DIVIDER
**When:** Between major sections of a deck. Creates rhythm and gives the audience a moment to reset.
**Content:** A single word or short phrase in Display/Mega. E.g. "The Opportunity." or "For MDs."
**Theme:** Dark.
**Rule:** The text should be a provocation or a promise, not a label. "Research" is a label. "What the Market Doesn't Know." is a promise.

---

### BIG TITLE
**When:** Opening a chapter. Introducing an idea that needs room.
**Content:** Page Title (top-left, uppercase) + Display/XL headline + 3-column body (each column: Eyebrow + Body text).
**Theme:** Light.
**Rule:** The headline is the hero. The three columns provide evidence, context, or sub-themes — not more headlines.

---

### TEXT INTRO
**When:** Narrative setup. When the idea needs a paragraph before it can breathe.
**Content:** Eyebrow + Body (max 200 words, 2 paragraphs) + Numbered List (3–6 items).
**Theme:** Light.
**Rule:** The numbered list is action-oriented. It names things, doesn't describe processes.

---

### 6-CARD GRID
**When:** Presenting 6 parallel items of equal weight. Evidence. Options. Modules. Exercises.
**Content:** Page Title + 3×2 cards, each with: Eyebrow/number + Title + Body (max 60 words).
**Theme:** Light.
**Rule:** Cards must be parallel. If one card has a number, all cards have numbers. If one has an eyebrow, all have eyebrows. No hierarchy within the grid.

---

### STATS
**When:** The numbers are the argument.
**Content:** Page Title + Display/XL headline + 3 stat callouts (each: Display/Mega number + 1-line label).
**Theme:** Light.
**Rule:** Numbers should be shocking, specific, and true. "47%" beats "nearly half." "£4.2B" beats "significant investment."

---

### 2×2 MATRIX
**When:** Mapping a 2-dimensional framework. Brand frameworks, market maps, tension grids.
**Content:** Page Title + Display/M title + 2×2 grid (each cell: Eyebrow + Display/M or Body).
**Theme:** Light.
**Rule:** The two axes should have genuine tension. If the quadrants feel obvious, the framework hasn't done its work.

---

### TODAY–TOMORROW
**When:** Contrast. Before/After. Current state vs. future state. Problem vs. solution.
**Content:** Page Title + 2 columns, each with: Eyebrow header + Display/M sub-headline + Body.
**Theme:** Light.
**Rule:** Left column is the friction. Right column is the relief. Left should feel uncomfortable to read.

---

### 4 RECOMMENDATIONS
**When:** Closing a section with clear actions.
**Content:** Page Title + 2×2 numbered cards, each with: Number (Display/M) + Overview (Eyebrow) + Actions (Body).
**Theme:** Light.
**Rule:** Recommendations are decisions, not suggestions. "Appoint a Brand Director" not "Consider brand leadership."

---

### SPLIT IMAGE + COPY
**When:** When an image does work that words cannot.
**Content:** Image (full-height, left half) + Copy (right half: Eyebrow + Display/M + Body).
**Theme:** Light (right panel) or Dark (right panel).
**Rule:** The image must be worth it. If it's stock, reconsider. If it's a client's product, it should be beautiful.

---

### POSITIONING STATEMENT
**When:** The brand idea, stated with conviction.
**Content:** Page Title + Display/XL quote (centre) + attribution (Futura PT, uppercase).
**Theme:** Light or Dark.
**Rule:** The statement should make someone feel slightly uncomfortable if they're doing it wrong, and relieved if they're doing it right.

---

### 3-COLUMN GRID
**When:** Three parallel themes, values, or principles.
**Content:** Page Title + Display/M title (optional) + 3 equal columns (each: Eyebrow + Body).
**Theme:** Light.
**Rule:** Three is the natural limit for parallel ideas. More than three belongs in a 6-Card Grid.

---

### 4-COLUMN GRID
**When:** Four items that are equal but distinct. Tools, steps, channels, formats.
**Content:** Page Title + 4 equal columns (each: Eyebrow + Title + Body).
**Theme:** Light.
**Rule:** Columns should scan left to right as a natural progression, not just a random collection.

---

### PROCESS CIRCLES
**When:** A sequence. The steps of a process, in order.
**Content:** Page Title + Display/M title + 3–6 circles in a row, connected by rules. Each circle: step name. Optional body below.
**Theme:** Light.
**Rule:** The circles are the content. Don't crowd them with too much text. If each step needs more than 10 words, use Text Intro instead.

---

### BRAND FRAMEWORK
**When:** MT's five-stage process or any pillar-based framework.
**Content:** Page Title + 5 pillars side by side, each with: Number/Name (Eyebrow) + Title + Body.
**Theme:** Light.
**Rule:** Pillars should feel sequential or parallel — not arbitrary. The framework should be legible in 30 seconds.

---

### TITLE + BODY
**When:** When the argument needs to be made in prose. Research findings, strategic synthesis.
**Content:** Page Title + Display/M headline + Body (up to 300 words, max 2 columns).
**Theme:** Light.
**Rule:** If you're using this template, the words must be worth reading. No bullet lists.

---

### BRAND PROFILE
**When:** Competitive landscape. A single brand, examined.
**Content:** Brand name in Display/XL + 2×2 attribute pairs (each: Eyebrow + Body).
**Theme:** Dark.
**Rule:** The brand name is the design. Keep it large. The four attributes should cover different dimensions: positioning, product, communication, culture.

---

### PERSONA BOARD
**When:** Consumer profiles. Research personas. Target audience.
**Content:** Image grid (left panel) + dark panel (right): Persona name + Display/M + pull quotes + attributes.
**Theme:** Mixed (light image, dark copy panel).
**Rule:** The persona should feel like a real person, not a demographic segment. "Katharina, 34, Hamburg" beats "Female, 30–40, urban."

---

### END SLIDE
**When:** Always the final slide.
**Content:** "MADE THOUGHT" logotype (or client-specific sign-off) + optional URL.
**Theme:** Dark.
**Rule:** No call to action. No email address. No QR code. The end slide is a statement of presence, not a sales tool.

---

## Template selection cheat sheet

| Primary content | Template |
|-----------------|----------|
| One big idea | Big Title |
| A number | Stats |
| A quote | Positioning Statement |
| A framework | 2×2 Matrix or Brand Framework |
| A sequence | Process Circles |
| A list of 6 | 6-Card Grid |
| A contrast | Today–Tomorrow |
| 4 actions | 4 Recommendations |
| An image | Split Image + Copy |
| An argument | Title + Body |
| A competitor | Brand Profile |
| A person | Persona Board |
| A chapter start | Section Divider (dark) |
| The beginning | Cover |
| The end | End Slide |
