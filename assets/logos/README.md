# MT Logo Assets

Add Made Thought logo SVG files to this folder.

## Required files

| Filename | Description |
|----------|-------------|
| `mt-logo-primary.svg` | Primary wordmark — black, for use on white backgrounds |
| `mt-logo-reversed.svg` | Reversed wordmark — white, for use on black backgrounds |
| `mt-logo-primary-stacked.svg` | Stacked variant (if available) |
| `mt-icon.svg` | Icon/monogram only (if available) |

## Usage rules

- **Light slides:** Use `mt-logo-primary.svg` (black)
- **Dark slides (covers, section dividers, end slides):** Use `mt-logo-reversed.svg` (white)
- **Minimum size:** Maintain legibility — wordmark should be readable at 32px height minimum
- **Clear space:** Maintain clear space equal to the cap-height of the "M" around all sides
- **Never:** Recolour, distort, add effects, or place on busy backgrounds without a clear panel
